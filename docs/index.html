Install

Use the release files from GitHub.

Download the latest release, run the Helper, then load the extension in your browser.

1

Start ClipTap Helper

Run ClipTapHelper.exe. The manager opens at http://127.0.0.1:17723.

2

Install the extension

Use the Firefox .xpi, or load the Chrome package as an unpacked extension.

3

Clip and download

Open a YouTube video, set the range, and send the request to the Helper.

FAQ

Things worth knowing.

Why does ClipTap need a Helper?

Browser extensions cannot directly run local programs like yt-dlp or FFmpeg. The Helper runs locally and performs the download work.

Does ClipTap upload videos to a server?

No. ClipTap is designed around a local Helper, so download requests are handled on your own computer.

Can it download full videos too?

Yes. The player panel includes both section download and full video download actions.

What about live streams?

Full video requests for live streams are shown as active recording instead of a normal fixed percentage progress bar.
