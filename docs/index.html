FAQ

Good to know

Small details before you install.

Why does ClipTap need a helper app?

Browsers cannot directly run yt-dlp or FFmpeg. The local helper receives requests and performs downloads on your machine.

Does it work offline?

The helper runs locally, but downloads still need access to the source website and any required video streams.

Is it open source?

Yes. ClipTap is built as an auditable local-first utility.
